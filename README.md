# GenZTech WhatsApp Bot

AI-powered WhatsApp lead generation & follow-up bot for GenZTech.

> Full README with setup steps, diagrams, and deployment instructions will be generated in Step 9.

## Quick local run
```bash
pip install -r requirements.txt
cp .env.example .env   # then fill in real values
python api/index.py
```

Server runs at `http://localhost:5000`.
